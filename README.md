# text-counter
A modern, real-time character and word counter built as a premium contact feature for the Greenden platform. This tool focuses on high-end UI/UX and live data processing.

🚀 Key Features
Live Character Tracking: Updates instantly as you type using JavaScript event listeners.

Word & Reading Time Logic: Calculates the word count and estimates the time it takes to read the message.

Unlimited Input: Unlike standard forms, this version allows for infinite text entry with a clean overflow scroll.

Glassmorphism UI: Features a "frosted glass" effect (backdrop-blur) over a lush nature-themed background.

Visual Status: Includes a pulsing "System Live" icon to confirm real-time script activity.

🛠️ Tech Stack
HTML5: Semantic input structure.

Tailwind CSS: Modern styling, glassmorphism, and responsive layouts.

Vanilla JavaScript: String manipulation and DOM updates.

Font Awesome: Elegant icons for the "System Live" and "Feather" status.

📖 How to Run
Save the code as index.html.

Open the file in any modern web browser.

Type in the textarea to see the counters update in real-time.
